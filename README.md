# typescript-node-env

## 使い方

ビルド(development build)

```
$ npm run dev
```

ビルド(production build)

```
$ npm run pro
```

ビルド(watch mode: ソースが更新されたら自動的に再ビルドするモード)

```
$ npm run watch
```

サーバーを開始する

```
$ node .dist\server.js
```
